# AstrBot 每日单词卡片插件

> 每天定时推送精美的多语种单词卡片，助力语言学习

## 一句话介绍

自动生成并推送玻璃拟态风格的单词卡片，支持英语六级、日语JLPT、成语、古文、无线电法规等 15,000+ 词条。

## 核心功能

- 🎨 **精美卡片**：玻璃拟态设计，4K级清晰度（1728×2160）
- 🌐 **多语种支持**：英语、日语、成语、古文、无线电法规
- 📅 **定时推送**：每天自动推送到群聊，时间可配置
- 📚 **丰富词库**：15,000+ 词条，随机/顺序学习
- 📊 **进度追踪**：自动记录学习进度，支持查看统计
- 🖼️ **CDN背景**：阿里云OSS加速，37张精选图片

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 安装渲染引擎 (Windows)
winget install wkhtmltopdf

# 注册推送
/vocab_register

# 切换卡组
/vocab_lang japanese

# 手动获取
/vocab
```
